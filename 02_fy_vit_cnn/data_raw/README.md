## 下载指南

| 类型 | 格式 | 数量 | 说明 |
|------|------|------|------|
| L1 级 FDI 产品 | `.HDF` | 64 个 | AGRI 一级数据（辐射定标） |
| L2 级 CLP 产品 | `.NC` | 64 个 | 二级云属性产品 |
